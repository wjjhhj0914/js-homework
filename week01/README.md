## 💭 회고

아직 JavaScript 문법에 완전히 익숙하지는 않지만,  
과제를 하면서 **직접 타이핑하고, 실행하고, 틀리고, "왜 안 되지?" 고민하고, 다시 고치는 과정**을 거치며 문법이 점점 **진짜 내 것이 되어간다는 느낌**을 받았다.
